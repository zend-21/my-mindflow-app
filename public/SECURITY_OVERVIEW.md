# 보안 개요

**셰어노트(ShareNote)**는 사용자의 개인정보와 데이터 보안을 최우선으로 생각합니다.

## 데이터 보호

### 로컬 우선 저장
- 모든 메모와 일정 데이터는 먼저 브라우저의 로컬 스토리지에 저장됩니다
- 인터넷 연결 없이도 앱을 사용할 수 있습니다

### 클라우드 동기화 (선택사항)
- Google 계정으로 로그인 시 Firebase Firestore에 데이터가 안전하게 백업됩니다
- 여러 기기에서 동일한 데이터에 접근할 수 있습니다

## 개인정보 보호

### 수집하는 정보
로그인 시 다음 정보만 수집합니다:
- 이메일 주소
- 이름 (Google 프로필)
- 프로필 사진 (선택사항)

### 수집하지 않는 정보
- 전화번호
- 주소
- 신용카드 정보
- 기기 정보 (최소한의 분석 데이터 제외)

## 암호화

### 전송 중 암호화
- 모든 네트워크 통신은 HTTPS로 암호화됩니다
- Firebase는 업계 표준 TLS/SSL을 사용합니다

### 저장 시 암호화
- Firebase Firestore는 자동으로 데이터를 암호화하여 저장합니다
- 시크릿 문서는 별도의 PIN으로 추가 암호화됩니다

## 접근 제어

### Firebase 보안 규칙
- 사용자는 본인의 데이터만 읽고 쓸 수 있습니다
- 친구 추가 시에도 명시적 동의가 필요합니다

### 세션 관리
- 로그인 토큰은 안전하게 관리됩니다
- 자동 로그아웃으로 보안을 강화합니다

## 데이터 소유권

### 사용자의 권리
- 언제든지 본인의 모든 데이터를 삭제할 수 있습니다
- 데이터 내보내기 기능으로 백업 가능합니다
- 계정 삭제 시 모든 데이터가 완전히 삭제됩니다

## 투명성

### 오픈소스
- 보안 관련 코드는 GitHub에서 공개됩니다
- 커뮤니티의 감사와 피드백을 환영합니다

### 정기 업데이트
- 보안 패치는 신속하게 배포됩니다
- 주요 변경사항은 사용자에게 공지됩니다
