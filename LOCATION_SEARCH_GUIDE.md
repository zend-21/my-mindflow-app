# 위치 기반 음식점 검색 가이드

## 기능 개요

음식점 검색 시 4가지 위치 모드를 선택하여 기준 위치를 설정할 수 있습니다.

## 네 가지 위치 모드

### 1. 현재위치
- GPS를 사용하여 현재 위치 기준으로 검색
- 모바일 앱에서는 정확한 GPS 위치 사용
- PC 웹에서는 IP 기반 위치 (정확도 낮음)
- 위치 권한이 필요합니다

**사용 방법:**
1. 리뷰 작성 페이지에서 "현재위치" 탭 클릭
2. 브라우저의 위치 권한 허용
3. 자동으로 현재 위치 기준 검색

### 2. 저장주소 슬롯 (주소1, 주소2, 주소3)
- **최대 3개**의 주소를 미리 저장해두고 빠르게 전환 가능
- 집, 회사, 자주 가는 곳 등을 각각 저장
- 저장된 주소 기준으로 5km 반경 내 음식점 검색
- 미설정 시 "미설정" 뱃지 표시

**리뷰 작성 페이지에서 빠른 설정:**
1. "주소1", "주소2", "주소3" 탭 중 하나 클릭
2. 미설정 상태면 자동으로 주소 입력 모달 표시
3. 주소 입력 (도로명, 지번 주소 또는 동/구 이름)
4. 검색 결과에서 원하는 주소 선택

**프로필 페이지에서 통합 관리:**
1. 프로필 페이지의 "저장 주소 관리" 섹션 접속
2. 3개 주소 슬롯을 한 번에 확인 및 관리
3. 각 주소마다 등록/변경/삭제 가능
4. 탭 라벨은 자동으로 지역명 표시 (예: "강남", "서초")

## 주소 검색 방법

### 상세 주소 입력
- **도로명 주소**: `강남구 테헤란로 123`
- **지번 주소**: `서초동 1234-5`
- 건물 번호까지 입력하면 더 정확한 위치 검색

### 간단한 입력
- **동/구 이름**: `서초동`, `강남구`
- 검색 결과에서 원하는 위치 선택

## 검색 반경

- 선택한 위치를 기준으로 **5km 반경** 내 음식점 검색
- 최대 **45개**의 검색 결과 표시
- 거리순으로 정렬되어 표시

## 데이터 저장

모든 위치 정보는 **로컬 스토리지**에 저장됩니다:
- `mindflow_saved_addresses`: 3개의 저장주소 (배열 형태)
- `mindflow_location_mode`: 마지막 선택한 위치 모드 (current, address1, address2, address3)

## 주의사항

1. **위치 권한**: 현재위치 모드 사용 시 브라우저 위치 권한 필요
2. **PC 위치 정확도**: PC 웹에서는 IP 기반 위치로 정확하지 않을 수 있음
3. **카카오 API 키**: 위치 검색 기능 사용을 위해 카카오 REST API 키 필요
4. **인터넷 연결**: 모든 위치 검색은 인터넷 연결 필요

## 문제 해결

### 주소 검색이 안 돼요
- 카카오 REST API 키가 `.env` 파일에 올바르게 설정되었는지 확인
- `KAKAO_API_KEY_SETUP.md` 파일 참고

### 현재위치 모드가 작동하지 않아요
- 브라우저 위치 권한 허용 확인
- 모바일 기기의 위치 서비스(GPS) 활성화 확인

### 검색 결과가 너무 적어요
- 검색 반경을 넓히려면 코드 수정 필요 (`radius` 값 조정)
- 카카오 API 제한으로 최대 45개까지만 표시
