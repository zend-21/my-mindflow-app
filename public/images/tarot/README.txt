RIDER-WAITE TAROT CARD IMAGES
===============================

This directory contains all 78 Rider-Waite tarot card images in public domain format.

SOURCE INFORMATION:
-------------------
- Primary Source: Wikimedia Commons (https://commons.wikimedia.org/wiki/Category:Rider-Waite_tarot_deck)
- Secondary Source: Sacred Texts Archive (https://sacred-texts.com/tarot/pkt/)
- All images are PUBLIC DOMAIN (original 1909 Rider-Waite-Smith deck)

-------------------

CARD BREAKDOWN:
-------------------
Major Arcana (22 cards):
  - major_00.jpg through major_21.jpg
  - 00=The Fool, 01=The Magician, 02=The High Priestess, etc.

Minor Arcana - Wands (14 cards):
  - wands_01.jpg through wands_14.jpg
  - 01=Ace, 02-10=numbered cards, 11=Page, 12=Knight, 13=Queen, 14=King

Minor Arcana - Cups (14 cards):
  - cups_01.jpg through cups_14.jpg
  - Same numbering as Wands

Minor Arcana - Swords (14 cards):
  - swords_01.jpg through swords_14.jpg
  - Same numbering as Wands

Minor Arcana - Pentacles (14 cards):
  - pentacles_01.jpg through pentacles_14.jpg
  - Same numbering as Wands

IMAGE SPECIFICATIONS:
-------------------
- Format: JPEG
- Resolution: Varies (300-1200 DPI)
- Dimensions: Approximately 300x530 to 1144x1919 pixels
- Color: RGB color mode
- Web-friendly sizes suitable for display

NAMING CONVENTION:
-------------------
The naming follows the pattern used in the Tarot.csv file:
- {suit}_{number}.jpg
- Numbers are zero-padded (e.g., 01, 02, 03)
- Suits: major, wands, cups, swords, pentacles
